th:each

⭐th:each 는 가장 많이 사용하는 태그로서 제일 중요⭐

이름 나이 성별 주소

th:each 는 두 번째 매개변수 stat 을 추가하게 되면 반복 상태를 확인할 수 있다.

        index : 0 부터 시작하는 인덱스 값
        count : 1 부터 시작하는 값
        size : 배열의 크기
        current : 현재 객체의 정보
        even : 짝수 번째 여부
        odd : 홀수 번째 여부
        first : 첫 번째 데이터 여부
        last : 마지막 데이터 여부
